1 """Simplify loading moveit config parameters.
2 
3 This module provides builder-pattern based class to simplify loading moveit related parameters found in
4 robot_moveit_config package generated by moveit setup assistant.
5 
6 By default it expects the following structure for the moveit configs package
7 
8 robot_name_moveit_config/
9  .setup_assistant -> Used to retrieve information about the SRDF file and
10  the URDF file used when generating the package
11  config/
12  kinematics.yaml -> IK solver's parameters
13  joint_limits.yaml -> Overriding position/velocity/acceleration limits from the URDF file
14  moveit_cpp.yaml -> MoveItCpp related parameters
15  *_planning.yaml -> planning pipelines parameters
16  cartesian_limits.yaml -> Pilz planner parameters
17  # TODO(JafarAbdi): Check to see if this is a good default value
18  robot_name_controllers.yaml -> trajectory execution manager's parameters
19  ...
20 
21 Example:
22  moveit_configs = MoveItConfigsBuilder("robot_name").to_moveit_configs()
23  ...
24  moveit_configs.robot_description
25  moveit_configs.robot_description_semantic
26  moveit_configs.robot_description_kinematics
27  moveit_configs.planning_pipelines
28  moveit_configs.trajectory_execution
29  moveit_configs.planning_scene_monitor
30  moveit_configs.move_group_capabilities
31  moveit_configs.joint_limits
32  moveit_configs.moveit_cpp
33  moveit_configs.cartesian_limits
34  # Or to get all the parameters as a dictionary
35  moveit_configs.to_dict()
36 
37 Each function in MoveItConfigsBuilder have a file_path as an argument which is used to override the default
38 path for the file
39 
40 Example:
41  moveit_configs = MoveItConfigsBuilder("robot_name")
42  # Relative to robot_name_moveit_configs
43  .robot_description_semantic(Path("my_config") / "my_file.srdf")
44  .to_moveit_configs()
45  # Or
46  moveit_configs = MoveItConfigsBuilder("robot_name")
47  # Absolute path to robot_name_moveit_config
48  .robot_description_semantic(Path.home() / "my_config" / "new_file.srdf")
49  .to_moveit_configs()
50 """
